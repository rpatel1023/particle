Stan Bot

  • A simple Discord Bot built with Discord.js

Stan Bot is a Discord Bot designed to perform basic commands with powerful flexibility. Built with Node JS, Stan Bot returns custom messages, accesses various APIs for fun chat features, and even return a user's recent Steam data. Stan Bot is designed to be easily modified, allowing users to add their own commands and enable/disable features to the bot.
